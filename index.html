Bevezetés

A responsive web design (RWB) alapú weboldal képes rugalmasan alkalmazkodni a különféle kijelzők méretéhez és felbontásához, könnyű olvashatóságot és egyszerű navigációt biztosítva a felhasználó számára. A reszponzív weboldal tehát minden eszközön az adott képernyő sajátosságaihoz igazodva jelenik meg.

A reszponzív weboldal készítése történhet egy adott sablon alapján is (például a WordPress-ben)vagy saját, egyéni megjelenésű oldalt is készíthetünk a HTML és a CSS megfelelő használatával.

Technikai szempontból a reszponzív webdesign három legfontosabb alapeleme a következő:

  • rugalmas, rács (grid) alapú elrendezés;
  • rugalmas, méretarányos képek és videók;
  • CSS media query.
Első lépés

Még mielőtt rátérnénk a reszponzív weboldalak fent említett három fontos elemére, először is meg kell említeni egy fontos beállítást a HTML fájlunkban.

Ez egy meta-tag, amelyet a head szekcióba kell beszúrjunk:

HTML

<meta name="viewport" content="width=device-width, initial-scale=1.0"/>

A viewport meta-tag utasításokat ad a böngészőnek az oldal méreteinek és méretezésének szabályozására.

  • A width=device-width úgy állítja be az oldal szélességét, hogy az kövesse az eszköz képernyőszélességét, ami eszköztől függően változik.
  • Az initial-scale=1.0 pedig beállítja a kezdeti nagyítási szintet.
Rácsos nézet

A legtöbb reszponzív weboldal alapja a rácsos nézet (grid-view), ami azt jelenti, hogy az oldal oszlopokra van osztva. Ez megkönyyíti a weboldal elemeinek az elhelyezését a különböző képernyők méreteivel arányos kivitelezésben. A reszponzív rácsos nézet néha akár 12 oszlopos is lehet, aminek a teljes szélessége 100%-os, és ami a böngészőablak átméretezésekor zsugorodik és bővül.

A rácsos nézetű megjelenéshez az első szükséges lépés a box-sizing beállítása a CSS-ben. A border-box tulajdonság hozzáadásával biztosítjuk, hogy a weboldal minden egyes eleme plusz keret (border) és margó (padding) nélkül jelenjen meg:

CSS

* { box-sizing: border-box; }

Ezután az oldalunkat a kívánt számú oszlopra bonthatjuk, megadva az egyes oszlopok szélességét a 100%-os teljes képernyőhöz viszonyítva. Két oszlop esetén például, ahol az egyik oszlopban a menü, a másikban pedig a tartalom jelenik meg, a következő kódolást használhatjuk:

.menu { width: 25%; float: left; } .main { width: 75%; float: left; }

Több oszlop esetén minden egyes oszlop szélességét meg kell határoznunk. Ha nem egy sórban szeretnénk az összes elemet elhelyezni, akkor nem szükséges, hogy az oszlopok szélességének az összértéke 100% legyen. Az egyes oszlopok szélességének az értéke az oldal egészéhez viszonyul, vagy ahhoz az elemhez, ami alá van rendelve a HTML fájlban. Így tehát egy 12 oszlopos nézetet a következőképp is feloszthatunk:

CSS

.col-1 {width: 8.33%;} .col-2 {width: 16.66%;} .col-3 {width: 25%;} .col-4 {width: 33.33%;} .col-5 {width: 41.66%;} .col-6 {width: 50%;} .col-7 {width: 58.33%;} .col-8 {width: 66.66%;} .col-9 {width: 75%;} .col-10 {width: 83.33%;} .col-11 {width: 91.66%;} .col-12 {width: 100%;}

Ha azt szeretnénk, hogy minden egyes oszlop egy irányba igazodjon, egyforma margóval és kerettel rendelkezzen, akkor ezeket a tulajdonságokat a * használatával egy kóddal is megadhatjuk, amennyiben a különböző .col-okat közös osztályba (class) soroltuk a HTML fájlunkban:

CSS

[class*="col-"] { float: left; padding: 15px; border: 1px solid red; }

Ha több oszloppal dolgozunk, akkor a HTML fájlunkban mindig meg kell adnunk, hogy az oszlopok hogyan rendeződnek el a sorokban (row). Amennyiben például .col-3-at (szélesseége: 25%) és a .col-9-et (szélesseége: 75%) egy sorban szeretnénk elhelyezni, akkor ezt a HTML fájlban kell eszerint elrendezzük:

HTML

<div class="row"> <div class="col-3">...</div> <div class="col-9">...</div> </div>

A soron belüli oszlopok "float" értékénél minden esetben "bal" (left) van megadva, ezért kikerülnek az oldal áramlásából, és a többi elem esetenként úgy kerülhet elhelyezésre, mintha az oszlopok nem is léteznének. Hogy ezt megakadályozzuk, meg kell adnunk egy pszeudo stílus tulajdonságot, amely törli ezt:

CSS

.row::after { content: ""; clear: both; display: table; }

Az eddigieket szemléltetve lássunk egy konkrét példát, hogyan épülhet fel egy RWD alapú weboldal, amely négy oszlopot tartalmaz:

HTML

<!DOCTYPE html> <html lang="hu"> <head>   <meta name="viewport" content="width=device-width, initial-scale=1.0">   <title>Magyarország</title>   <link rel="stylesheet" href="styles.css"> <head> <body>   <div class="col-1 header">      <h1>Magyarország</h1>   </div>   <div class="row">      <div class="col-2 menu">         <ul>            <li>Államforma</li>            <li>Határos országok</li>            <li>Népesség</li>            <li>Nyelv</li>            <li>Főváros</li>         </ul>      </div>      <div class="col-3">         <h1>Magyarország</h1>         <p>Magyarország állam Közép-Európában, a Kárpát-medence közepén. 1989 óta parlamentáris köztársaság.</p>         <p>Északról Szlovákia, északkeletről Ukrajna, keletről és délkeletről Románia, délről Szerbia és Horvátország, délnyugatról Szlovénia, nyugatról pedig Ausztria határolja.</p>         <p>Népessége a 21. században 10 millió alá csökkent.</p>         <p>Hivatalos nyelve a magyar, amely a legnagyobb az uráli nyelvcsaládba tartozó nyelvek közül.</p>         <p>Fővárosa és legnépesebb városa Budapest.</p>      </div>   </div>   <div class="col-4 footer">      <p>Forrás: <a href="https://hu.wikipedia.org/ wiki/Magyarország">Wikipédia</a></p>   </div> </body> </html>

CSS

* { box-sizing: border-box; } html, body { font-family: "Lucida Sans", sans-serif; background-color: beige; } [class*="col-"] { float: left; padding: 15px; } .col-1 {width: 100%;} .col-2 {width: 30%;} .col-3 {width: 70%;} .col-4 {width: 100%;} .row::after { content: ""; clear: both; display: table; } .header { background-color: #CE2939; color: #ffffff; padding: 15px; } .menu ul { list-style-type: none; margin: 0; padding: 0; } .menu li { padding: 10px; margin-bottom: 10px; background-color: #477050; color: #ffffff; cursor: pointer; } .menu li:hover { background-color: #CE2939; } .footer a { color: #477050; } .footer a:hover { color: #CE2939; }
Reszponzív képkezelés

Hogy a képek reszponzív módon jelenjenek meg fontos, hogy a szélesség 100%-ra legyen állítva, amelyhez alkalmazkodik a magasság:

CSS

img { width: 100%; height: auto; }

Mivel azonban ez a beállítás a képet a képernyőhöz igazítva az eredeti méreténél nagyobbra is méretezheti, sok esetben jobb megoldás a "max-width" tulajdonság használata. Így a kép méretei az eredeti méretéhez képest csökkenhetnek, ha szükséges, de nem nem lehet nagyobb, így a kép eredeti felbontásának a minősége nem csökken:

CSS

img { max-width: 100%; height: auto; }

A háttérképek ugyancsak alkalmazkodhatnak a felhasználó által használt képernyő méretéhez. Ennek érdekében a kép eredeti méretétől függően három módszert használhatunk:

  • Ha a "background-size" tulajdonság "contain" értékre van állítva, akkor a háttérkép megpróbál illeszkedni a tartalmi területhez. A kép azonban megtartja az arányait:

    CSS

    div { width: 100%; height: 500px; background-size: contain; }

  • Ha a background-size tulajdonság "100% 100%" értékre van állítva, a háttérkép a teljes tartalmi területre kiterjed:

    CSS

    div { width: 100%; height: 500px; background-size: 100% 100%; }

  • Ha a "background-size" tulajdonság "cover" értékre van állítva, akkor a háttérkép úgy méreteződik, hogy a teljes tartalmi területet lefedje, megtartva a kép eredeti arányait; ebben az esetben a kép egy része eltűnhet:

    CSS

    div { width: 100%; height: 500px; background-size: cover; }

Ha hozzáadunk a "Magyarország" weboldalunkhoz egy képet, például Magyarország címerét, amelyet a fejléc jobboldalán helyezünk el, akkor a fentebbi kódunkat a következőképp egészíthetjük ki:

HTML

<div class="col-1 header">   <h1>Magyarország   <div class="coat-arms">      <img src="(url)" alt="coat of arms hungary">   </div> </div>

CSS

.header { background-color: #CE2939; color: #ffffff; padding: 10px; display: flex; } h1 { padding: 15px 15px; margin-right: auto; } img { max-width: 100%; height: 100px; background-size: cover; }
Media query

A "media query" a CSS3-ban bevezetett technika, amely a "@media" tulajdonság használatával szabályozza, hogy a weboldal egyes elemeinek a tulajdonságai, milyen értékeket kapjanak a különböző képernyőszélességek függvényében. Például megadhatjuk, hogy a háttérszín 600px szélesség alatt kék legyen, míg afölött továbbra is fekete marad:

CSS

body { background-color: black; } @media only screen and (max-width: 600px) { body { background-color: blue; } }

A "@media" tulajdonság segítségével a különböző képernyők méretéhez szabhatjuk az oszlopok és képek elrendezését és megjelenését is is. A korábban létrehozott "Magyarország" oldalt például a jobb mobilmegjelenés érdekében így módosíthatjuk:

CSS

@media only screen and (max-width: 600px) { /* Extra small devices: */ .col-1 { width: 100%;} .col-2 { width: 100%; text-align: center;} .col-3 { width: 100%;} .col-4 { width: 100%; text-align: center;} img { display: none; } }

Az így elkészült RWD alapú weboldalt lásd itt.
